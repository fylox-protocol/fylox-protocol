# Fylox Integrations

Fylox is designed as a lightweight payment infrastructure layer that enables developers, merchants and applications to integrate real-world Pi payments into their products.

The protocol begins with a simple QR-based payment flow and evolves toward a broader payment infrastructure for the Pi ecosystem.

---

# Integration Overview

Applications and services can integrate Fylox to enable Pi payments in real-world and digital environments.

Typical integration flow:

User → Scan QR → Fylox Payment Layer → Pi Network Transaction → Merchant Confirmation

Fylox simplifies the interaction between applications, merchants and the Pi Network blockchain.

---

# Current Integration (MVP)

The first version of Fylox focuses on a minimal integration designed for testing real-world payment flows.

Features available in the MVP:

• QR-based Pi payments  
• Peer-to-peer payment confirmation  
• Merchant payment reception  
• Basic transaction validation  

The MVP is designed primarily for usability testing within the Pi ecosystem.

---

# Merchant Integrations

Merchants can integrate Fylox to accept Pi payments through QR codes.

Example use cases:

• Local businesses  
• Restaurants and cafés  
• Retail stores  
• Service providers  
• Independent sellers  

The integration allows merchants to receive Pi payments directly from pioneers.

---

# Application Integrations

Developers can integrate Fylox into applications that require payment functionality.

Possible integrations include:

• Marketplaces  
• Delivery apps  
• Ticketing platforms  
• Transport services  
• Digital services  

Fylox acts as a payment layer connecting applications with Pi transactions.

---

# Future Integrations

Future versions of Fylox may expand integration capabilities including:

• Developer APIs  
• SDK support

• Payment analytics  
• Expanded payment infrastructure modules  

These tools will allow developers and businesses to build payment-enabled applications on top
