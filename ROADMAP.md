# Fylox Protocol Roadmap

Fylox is building the real-world payment infrastructure for the global Pi Network ecosystem.

Our goal is simple: enable Pi to function as a real-world currency for everyday transactions.

Below is the current development roadmap.

---

# Phase 1 — Protocol Foundation

Launch the initial Fylox prototype and infrastructure.

Key milestones:

• Public website and protocol introduction  
• Initial prototype deployed via GitHub Pages  
• Whitepaper publication  
• Open-source repository launch  
• Community feedback and early contributor onboarding  

Status: In progress

---

# Phase 2 — Developer Ecosystem

Open the protocol to builders and developers.

Key milestones:

• Developer documentation  
• API and integration guidelines  
• Community contributions via GitHub  
• First experimental integrations with Pi Browser  
• Architecture documentation (Oracle, Nexus, Agora layers)

Goal: enable developers to build on top of Fylox.

---

# Phase 3 — Merchant Infrastructure

Enable real-world usage of Pi through merchant adoption.

Key milestones:

• QR-based payment system  
• Merchant onboarding framework  
• Merchant dashboard prototype  
• Pilot integrations with local businesses  
• First real-world payment experiments

Goal: allow pioneers to spend Pi in real businesses.

---

# Phase 4 — Network Expansion

Scale the Fylox ecosystem globally.

Key milestones:

• International merchant integrations  
• Payment infrastructure improvements  
• Reputation layer (Nexus) deployment  
• Governance framework (Agora)  
• Expanded developer ecosystem

Goal: build a global network for Pi payments.

---

# Phase 5 — Global Infrastructure

Establish Fylox as a core payment layer within the Pi ecosystem.

Key milestones:

• Large-scale merchant network  
• Advanced protocol integrations  
• Decentralized governance participation  
• Global adoption initiatives

Goal: make Pi usable for everyday payments worldwide.

---

# Vision

Fylox aims to become the infrastructure layer that enables real-world utility for Pi.

Millions of pioneers already hold Pi.

The next step is simple:

Give them a place to spend it.

Built for the global community of Pi Pioneers.
