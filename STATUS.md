# Fylox Protocol Status

Last updated: 2026

## Project Stage

Fylox Protocol is currently in **early development**.

The project is focused on building the initial infrastructure and testing basic payment interactions using Pi Network.

## Current Focus

The current development stage focuses on:

- Building the initial Fylox protocol architecture
- Developing a minimal payment flow
- Testing QR-based Pi payment requests
- Allowing early pioneers to test transactions in controlled environments

This stage is designed to validate the core concept before expanding the protocol.

## MVP

The first version of Fylox will be a **Minimal Viable Product (MVP)** that allows:

- Basic payment requests
- QR-based payment interactions
- Simple merchant testing
- Pioneer testing inside the Pi ecosystem

This MVP will be submitted to the Pi developer ecosystem for early experimentation and feedback.
